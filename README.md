# GFC Institutions Website
A beginner-friendly website for **GFC Group of Institutions**, showcasing programs, about section, and an application form. Built using **HTML & CSS**.  
# Features
- Header with college name and call-to-action button  
- About section describing the college  
- Why Choose Us section highlighting key benefits  
- Programs section listing UG & PG courses  
- Apply page with simple student form  
- Responsive design for mobile & desktop
